## 1교시 — Part 1. 아이스브레이커: Draw Toast (15분)

> **1교시 운영안**: Part 0 (10분) + **Part 1 Draw Toast (15분)** + Part 2 Mermaid 입문 (25분) = 50분  
> 참고: [Draw Toast — Gamestorming](https://gamestorming.com/draw-toast/)

### 1.0 왜 "토스트"인가? — 게임 목적 (2분)

> 강사가 **슬라이드 1장**으로 설명. 거창한 용어(디자인 씽킹, 시스템 사고)는 **쓰지 않습니다**.

**교육생에게 읽을 멘트 (요지)**

> *"잠시 후 Mermaid, flowchart, diagram-spec… 어려운 말이 나옵니다.*  
> *그 전에, **왜 다 같이 토스트 그림을 그리는지** 아주 쉽게 말씀드릴게요."*

**목적 ① — "말로는 다 아는데, 생각은 다 다르네?"**

회의에서 *"토스트 만들면 되죠?"* 하면 모두 고개를 끄덕입니다.  
그런데 **그림**을 그리라고 하면 결과가 완전히 달라집니다.

| 사람 | 그리는 것 | 집중 |
|------|-----------|------|
| **A** | 토스터기 → 식빵 → 잼 | **과정** |
| **B** | 프라이팬·버터·식빵 굽기 | **방법** |
| **C** | 밀밭 → 공장 → 마트 → 집 → 토스터 | **스케일** |

→ **같은 단어 "토스트"를 써도, 머릿속 그림의 범위·깊이는 사람마다 다릅니다.**  
→ 오늘 오후 **"출장 정산 업무"** 흐름도를 그릴 때도 **똑같이** 일어납니다.

**목적 ② — 복잡한 일 전, 두뇌 스트레칭**

*"앱 결제 시스템 고쳐봅시다!"* → 입 다물어짐.  
*"토스트 만드는 법 그려보세요"* → 손이 움직이고 웃음이 납니다.  
**손·대화를 먼저 풀어 주는 워밍업**입니다.

**목적 ③ — (슬라이드에는 한 줄만) "박스와 화살표" 맛보기**

토스트 그림에는 식빵, 토스터, 손, 잼… **여러 요소**가 **선으로 연결**됩니다.  
오늘 배울 **업무 흐름도**도 *"무엇(산출물)"* 과 *"일(프로세스)"* 을 **화살표로 잇는 것**과 같습니다.

**한 줄 요약 (슬라이드 하단)**

> 💡 *"겉으로는 같은 말을 해도 머릿속 생각은 다르다 — 복잡한 일 전에 **그림으로 꺼내 놓고 싱크** 맞추자!"*

### 1.1 규칙 (1분)

| ✅ | ❌ |
|---|---|
| **그림만** 그리기 | 글·숫자·화살표 글자 쓰기 |
| A4·포스트잇·펜 | ChatGPT / Gemini |
| 낙서·简笔画 OK | 남 그림 비판 |
| 2~3분 안에 완성 | 완벽한 그림 |

> *"미술 잘 못 그려도 됩니다. **낙서**가 더 좋을 때도 있어요."*

### 1.2 진행 시나리오 (10~12분)

**① 준비 (1분)**

- 교육생에게 A4 또는 포스트잇 + 펜 1세트씩
- 강사도 **종이 1장** 준비 *(선택: 본인도 같이 그리면 분위기 ↑)*

**② 그리기 (2~3분)**

> 강사가 **타이머**를 켜고 **그대로 읽어도 되는 멘트**:

> *"**말이나 글자는 쓰지 마세요.**  
> **오직 그림으로만** '토스트 만드는 법'을 그려 보세요.  
> 토스트를 한 번도 안 만들어 본 사람에게 가르친다고 생각하시면 됩니다.  
> **3분** — 시작!"*

- 타이머 종료 → *"연필 내려놓으세요. 안 끝났어도 괜찮습니다."*

**③ 붙이고 비교 (4~5분)**

- 교육생 그림을 **벽·보드·책상 한쪽**에 쭉 붙임 (마스킹테이프)
- 전체가 **한 바퀴 구경** (1~2분)
- 강사가 **2~3장** 짚어서 질문 *(이름 부르지 않고 "이 그림", "저 그림"도 OK)*:

  - *"어디서 **시작**했나요? 식빵? 토스터? 밭?"*
  - *"**화살표**나 순서를 표시했나요?"*
  - *"**가장 많은 요소**가 그려진 그림은?"*

**④ 이야기 (4~5분)**

> 아래는 **대화 촉진용 질문** — 2~3개만 골라도 됨.

1. *"토스터기만 그린 분? 프라이팬부터 그린 분?"*
2. *"재료(식빵·잼)만 그리고 **과정**은 생략한 분?"*
3. *"밀·공장·마트까지 그린 **가장 긴** 그림 — 무슨 생각이었나요?"*
4. *"다른 사람 그림 보면서 **'어? 나랑 다르네'** 한 순간?"*

**기록 (교육생, 선택)**

| 칸 | 적기 |
|---|------|
| 내 그림의 **시작점** | (식빵 / 토스터 / …) |
| **요소 개수** (대략) | (   )개 |
| 남 그림과 **가장 달랐던 점** | (한 줄) |

**⑤ 2회차 본론으로 연결 (1분)**

> **종료 멘트 (그대로 읽어도 OK)**:

> *"보셨죠? **'토스트' 한 마디**인데 그림은 전부 다릅니다.*  
> *오늘 오후에는 **여러분 업무** — 출장 정산이든, 회의록이든 — 를 **이렇게 그림으로** 펼칩니다.*  
> *다만 손 대신 **컴퓨터가 읽는 텍스트**(Mermaid)로 그릴 거고, **팀이 공유할 규칙**(diagram-spec)도 씁니다.*  
> *아까 토스트에서 **박스랑 화살표** 쓰신 분 — 이미 반은 하신 겁니다. ☕"*

**▶ 자주 나오는 그림 유형 (비교용으로 입에 올릴 말)**

| 유형 | 특징 | 한마디 |
|------|------|--------|
| **미니멀** | 식빵 + 토스터 2개 | "핵심만!" |
| **요리사** | 프라이팬·버터·불 | "방법이 중요" |
| **시스템** | 밭→공장→마트→… | "전체 연결" |
| **사용자** | 사람 손·입·접시 | "누가 먹나" |

**▶ 막힐 때**

- 그리기가 너무 느리면 → **2분**에 종료
- 붙일 벽이 없으면 → **책상 위에 펼쳐** 두고 자리만 돌아다니기
- 분위기 침체되면 → *"가장 **화려한** 그림 / **가장 단순한** 그림"* 손들어 보기

**▶ 하지 말 것**

- Draw Toast 중 **ChatGPT·Mermaid 시연** — 게임과 섞지 않음 (Part 2에서)
- "디자인 씽킹", "시스템 사고" **용어 남발**
- 특정 교육생 그림 **평가·순위** 매기기

**▶ Part 2 (Mermaid) 첫 슬라이드 연결 멘트**

> *"아까 토스트, 기억나시죠? 이제 **같은 일**을 키보드로 합니다. [mermaid.live](https://mermaid.live/) 열어 주세요."*
