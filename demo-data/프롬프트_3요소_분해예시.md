# 프롬프트 3요소 분해 예시 — 출장 정산

## 원본 MD (한 덩어리)

```
[역할] (주)멋진엔지니어링 총무 담당
[작업] 아래 영수증 목록으로 출장 정산서 표 작성
[출력] | 일자 | 항목 | 금액 | 비고 | + 합계
[데이터]
(영수증 붙여넣기)
```

## 3요소로 분해

| 요소 | 해당 내용 |
|------|-----------|
| **Input** | 영수증.txt, 출장계획서.md (참조) |
| **Instruction** | 역할 + 작업 + 회사 양식 규칙 |
| **Output** | 표 형식 + 합계 행 |
