## 사용 원칙 (지시서 §3.2)

1. **순서 일관**: 회계 → 기획 → 총무 → 업무 (학생 부서 비중에 따라 바꾸지 않음)
2. **라이브 시연**: **회계부** 1개만 (가장 깔끔하게 입출력 분리 + 5노드)
3. **본인 부서 모방**: 학생이 본인 부서 카드를 보면서 자기 1차 프롬프트 작성
4. **부서 카드 없는 학생**: AI 인터뷰(1교시)에서 본인이 직접 후보 발굴
