# 출장 정산서 작성기 — HTML 마이크로 도구 사양서 (견본)

> micro-tool-interviewer 결과 예시 · (주)멋진엔지니어링 가공

## 개요

영수증 목록(CSV/텍스트)과 출장 계획서를 붙여 넣으면, 회사 양식에 맞는 **출장 정산서 표**를 자동 생성한다.

## 입력

- 파일 형식: `.txt` 또는 `.csv` (영수증), `.md` (출장계획서, 참조)
- 데이터 구조: 일자, 항목, 금액, 비고
- 예시: `demo-data/출장정산_입력샘플/영수증.txt`

## 처리 과정

1. 영수증 텍스트를 줄 단위로 파싱
2. 항목별 합계·총액 계산
3. 출장계획서의 기간·목적을 헤더에 반영
4. 정산서 표 형태로 출력

## 출력

- 형식: 화면 표 + 클립보드 복사용 텍스트
- 구조: | 일자 | 항목 | 금액 | 비고 | + 합계 행

## UI 구성

- 영수증 입력 textarea
- (선택) 출장계획서 textarea
- [정산서 생성] 버튼
- 결과 표 + [복사] 버튼

## 예외 처리

- 빈 줄·잘못된 형식 → 해당 줄 빨간색 표시
- 금액이 숫자가 아니면 경고

## 기술 참고

- 단일 HTML 파일
- CDN: (필요 시) PapaParse for CSV
